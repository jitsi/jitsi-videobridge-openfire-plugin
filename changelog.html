Jitsi Video Bridge Plugin Changelog

2.6 -- tbd

  • Class loading mechanism made somewhat more flexible.

2.5 -- June 25, 2018

  • Improve configuration options through admin console.
  • Align the port range definition used to dynamically assign media stream ports to the definition used in JVB (was: 5000-6000 UDP, now is 10001-20000 UDP).
  • Changes to the port configuration are now actually applied (although require a restart).
  • Exposes Jitsi Videobridge single port configuration in admin console (used when multiplexing media streams).
  • Updated JVB dependency from 1.1-20180307.203301-38 to 1.1-20180621.193237-72.
  • Now used Openfire 4.2.0 (instead of 4.2.0-SNAPSHOT) as it's API source.

2.4 -- March 14, 2018

  • Updated JVB dependency from 1.1-20180130.233151-31 to 1.1-20180307.203301-38.

2.3 -- January 31, 2018

  • Updated JVB dependency from 1.1-20171031.202356-10 to 1.1-20180130.233151-31.

  • Updated JVB dependency to 1.1-20171031.202356-10.
